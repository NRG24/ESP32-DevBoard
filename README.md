# ESP32-DevBoard
A classic, but beefy ESP32 DevBoard with WiFi and Bluetooth Antenna integrated. There are random I2C sensors such as a Max PPG sensor and a small IMU. There is also power path management with a PMOS that switches between battery power and VBUS. There is a lot of diode ESD protection.
